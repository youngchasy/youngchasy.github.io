# Youngchasy portfolio project

## Что внутри

- `index.html` — сам сайт
- `editor.html` — редактор контента без правки кода
- `content.json` — все тексты, ссылки и карточки
- `assets/images/` — сюда складывай картинки
- `assets/audio/` — сюда складывай аудио

## Что изменилось

- верхний экран стал чище
- правые два блока объединены в один
- убраны лишние подписи и плашки
- убрана кнопка `Редактор` из верхнего меню сайта
- добавлен тёмный фоновый арт с параллакс-эффектом
- улучшена мобильная адаптация
- данные верхних блоков теперь тоже редактируются через `editor.html`

## Как менять тексты без кода

### Самый удобный способ
1. Запусти локальный сервер.
2. Открой `editor.html`
3. Меняй нужные поля
4. Нажми **Сохранить черновик**
5. Нажми **Открыть предпросмотр**
6. Когда всё нравится — нажми **Скачать content.json**
7. Замени старый `content.json` новым

## Как менять верхние блоки

Теперь через редактор можно менять:
- название
- домен
- главный заголовок
- подзаголовок
- текст кнопок
- фоновую картинку
- весь правый верхний блок:
  - заголовок
  - текст
  - 3 статы
  - заголовок списка
  - 4 пункта списка

## Как менять описание карточек работ

Открой `editor.html`.

В разделе **Карточки работ** у каждой карточки можно отдельно менять:
- заголовок
- категорию
- описание
- ссылку
- текст кнопки
- картинку
- аудио

## Как добавлять свои картинки и аудио

### Картинки
Клади файлы в папку:
`assets/images/`

Пример:
`assets/images/my-work-01.jpg`

В редакторе вставляй путь:
`./assets/images/my-work-01.jpg`

### Аудио
Клади файлы в папку:
`assets/audio/`

Пример:
`assets/audio/beat-preview-01.mp3`

В редакторе вставляй путь:
`./assets/audio/beat-preview-01.mp3`

## Как запускать локально

### Вариант 1 — через VS Code
1. Установи VS Code
2. Установи расширение **Live Server**
3. Открой папку проекта
4. Открой `index.html` или `editor.html` через **Open with Live Server**

### Вариант 2 — через Python
Если Python уже установлен:

1. Открой папку проекта
2. В адресной строке папки введи `cmd`
3. В консоли выполни:

```bash
python -m http.server 5500
```

4. Потом открой:
- `http://localhost:5500/index.html`
- `http://localhost:5500/editor.html`

## Как заливать на хостинг

Если используешь Cloudflare Pages:
1. Загружаешь весь проект целиком
2. При изменениях обновляешь `content.json`
3. Перезаливаешь проект

## Важно

Черновик в `editor.html` сохраняется в браузере через localStorage.
Это удобно для предпросмотра, но это не заменяет файл `content.json`.

Чтобы изменения реально попали на сайт:
- скачай новый `content.json`
- замени им файл в проекте
- залей проект на хостинг
